# 新闻追踪技能

## 目录结构

```
news/
  daily/YYYY-MM-DD.md     ← 当天原始新闻存档（只追加，不修改）
  topics/
    伊朗战争.md           ← 话题汇总（持续更新）
    英国制造业.md
    海水淡化.md
    ...
```

## 核心原则

1. **英文标题 → 存档，不展示给用户**
2. **展示给用户 = 中文翻译 + 关键事实**
3. **按话题聚合，不按时间碎片化**

---

## 新闻处理流程

### 1. 收到新新闻（来自早报/追踪）

- 英文标题 + 链接 → 写入 `daily/YYYY-MM-DD.md`（追加）
- 判断话题 → 更新对应 `topics/XX.md`

### 2. 话题文件结构

```markdown
# [话题名称]

## 背景（Wikipedia 摘要）
> 来自 Wikipedia 的关键事实

## 时间线
- 2026-03-XX：事件描述
- 2026-04-01：最新动态

## 关键事实
- 事实1
- 事实2

## 追踪状态
- 阶段：爆发期 / 持续 / 消退
- 重要程度：高 / 中 / 低
```

### 3. 更新话题文件

- 新条目 → 插入到「时间线」顶部
- 状态变化 → 更新「追踪状态」
- 新关键事实 → 追加到「关键事实」

### 4. Wikipedia 参考

- 英文：https://en.wikipedia.org
- 中文：https://zh.wikipedia.org
- 查背景事实时访问 Wikipedia，不预先存档

---

## 英文标存档格式（daily/）

```markdown
## 2026-04-03

- [标题] https://链接 (高/中/低重要)
- [标题] https://链接 (高/中/低重要)
```

---

## 输出给用户的格式

**中文翻译 + 关键事实**，不要直接甩英文标题。

---

## 示例

### 输入（英文新闻）
```
Iran war snarls supply chains impacting factory costs
最后更新: 2026-04-01 | 阶段: outbreak
```

### 存档（daily/2026-04-03.md）
```
- Iran war snarls supply chains impacting factory costs | https://... (高)
```

### 话题更新（topics/伊朗战争.md）
```markdown
## 时间线
- 2026-04-03：新闻追踪新增——伊朗战争导致供应链混乱，影响工厂成本

## 关键事实
- 伊朗战争影响全球供应链
- 工厂成本上升
```

### 展示给用户
```
🔴 伊朗战争导致供应链混乱，影响工厂成本
- 来源：追踪链 2026-04-03 | 阶段：爆发期
```
