# 财务摘要 → ERPNext 规范化映射

## 观察数据基础

样本来源：吉众历史账单（2019-2025，银行/现金/应付账款各sheet）

---

## 一、摘要结构类型

### 类型A：【供应商-物料名】

```
物业-电费        → 供应商=物业，物料名=电费
物业-自来水      → 供应商=物业，物料名=自来水
员工-工资取现金  → 供应商=员工，物料名=工资取现金
员工-社会保险    → 供应商=员工，物料名=社会保险
员工-住房公积金  → 供应商=员工，物料名=住房公积金
银行-手续费      → 供应商=银行，物料名=手续费
银行-短信费      → 供应商=银行，物料名=短信费
金普金-详见明细  → 供应商=金普金，物料名=详见明细（→ 备注补充）
```

**规律**：`供应商-物料` → 拆分为 supplier + item_name

---

### 类型B：【物料+规格型号】（空格分隔）

```
浸渗剂  95-1000AA      → item_name=浸渗剂，规格=95-1000AA
B类随行记录卡-F版       → item_name=B类随行记录卡，规格=F版
不锈钢止回阀            → item_name=不锈钢止回阀，规格=空
螺丝                     → item_name=螺丝，规格=空
劳保鞋                   → item_name=劳保鞋，规格=空
车用油-汽油              → item_name=车用油-汽油（可拆：车用油，规格=汽油）
车用油-柴油              → item_name=车用油-柴油
货物运输费               → item_name=货物运输费，规格=空
```

**规律**：
- 有空格分隔 → 第一个空格前是 item_name，后面的空格后是规格
- 规格在物料名后用空格分隔 → item_name 保留物料部分，规格独立

---

### 类型C：【含规格的物料-供应商】混合

```
浸渗剂  95-1000AA  → item_name=浸渗剂，规格=95-1000AA，备注=浸渗DL382使用
硒鼓（激光多功能一体机） → item_name=硒鼓，备注=brotjer激光多功能一体机用
```

**规律**：规格型物料的供应商/用途信息 → 写入备注

---

### 类型D：【无规格的纯物料】（无供应商前缀）

```
招待餐费           → item_name=招待餐费
压力表检验         → item_name=压力表检验
车间-工作餐餐费    → item_name=车间-工作餐餐费（车间-前缀保留）
技术服务费          → item_name=技术服务费
环保检测服务费      → item_name=环保检测服务费
劳务费              → item_name=劳务费
备用金              → item_name=备用金
年终奖备用金        → item_name=年终奖备用金
```

**规律**：无供应商前缀，直接用摘要作为 item_name

## 四、规范化规则总结

### 4.1 item_name 命名规则（优先级从高到低）

1. **工序/费用类**：直接用工序名（电费、水费、手续费、招待餐费）
2. **原材料类**：用物料名（螺丝、劳保鞋、不锈钢止回阀）
3. **带规格类**：物料名 + 空格 + 规格（浸渗剂 95-1000AA）
4. **带品牌类**：保留品牌（硒鼓（激光多功能一体机）→ 硒鼓）

**禁止写入 item_name 的内容：**
- ❌ 供应商前缀（物业-、员工-、银行-）→ 应写入 supplier_name 或备注
- ❌ 期间说明（2024年12月、2025年01月）→ 应写入备注
- ❌ 用途说明（浸渗DL382使用）→ 应写入备注

---

### 4.2 custom_备注 写法规则

| 场景 | 备注内容 | 示例 |
|------|---------|------|
| 公共事业费 | 归属期间+费用类型 | 2024年12月电费 |
| 原材料用途 | 使用对象/设备+具体用途 | 浸渗DL382使用 |
| 招待费 | 招待对象+事由 | 招待诺玛科乔工午餐费用 |
| 运输费 | 出发地→目的地+事由 | 北京奔驰到吉众货物运输费用 |
| 车辆费 | 归属期间+油品 | 2025年01月汽油 |
| 设备维修 | 设备+维修内容 | 真空泵维修费 |
| 工资/社保 | 归属月份+类型 | 2024年11月工资 |
| 无需备注 | 可留空 | 备用金、技术服务费 |

---

### 4.3 custom_guige_xinghao 规则

| 场景 | 规格写法 | 示例 |
|------|---------|------|
| 化学品/材料 | 型号/货号 | 95-1000AA、F版 |
| 油品 | 油品类型 | 汽油、柴油 |
| 五金/耗材 | 通常不留规格 | 螺丝、劳保鞋（按实际） |
| 混合型 | 规格+类型 | 车用油-汽油 → item_name=车用油，规格=汽油 |

## 五、特殊案例处理

### 5.1 供应商-物料 分离

```
摘要：物业-电费
→ supplier_name = 物业（或"物业公司"）
→ item_name = 电费
→ custom_备注 = 2024年12月电费

摘要：员工-社会保险
→ supplier_name = 员工（或"社保局"）
→ item_name = 社会保险
→ custom_备注 = 2025年01月社会保险
```

### 5.2 同物料多用途

同一物料（如"螺丝"）在不同行有不同用途 → 备注分开填写，不合并。

### 5.3 无供应商前缀的"供应商-物料"混合词

```
摘要：车间-工作餐餐费
→ item_name = 车间-工作餐餐费（整体保留）
→ custom_备注 = 2024年12月员工工作餐餐费
```

### 5.4 "详见明细"类

```
摘要：金普金-详见明细202408202412
→ item_name = 金普金
→ custom_备注 = 2024年8-12月明细汇总（补充完整期间）
```

## 六、工作流程

1. **读取原始财务数据**（xls/xlsx 的银行/现金 sheet）
2. **识别摘要类型**（A/B/C/D 四种）
3. **提取 item_name**：去掉供应商前缀、保留物料和规格
4. **提取 custom_guige_xinghao**：从物料名中空格分隔的部分
5. **生成 custom_备注**：按「期间+用途+供应商」组合
6. **核对无误后** → 输出标准格式用于 ERPNext 录入或匹配
