# 精益思想 — 章节笔记

## 五大原则

### 原则1：定义价值（Define Value）
- 价值只能由**最终客户**确定
- 以客户的眼光定义什么是"价值"
- 价值 = 以正确价格、在正确时间、提供正确产品

### 原则2：识别价值流（Identify the Value Stream）
- 价值流 = 使一件产品通过三项管理任务的所有活动
  - 解决问题
  - 信息管理
  - 物质转化
- 识别三类活动：
  1. **创造价值的步骤** ✓
  2. **不创造价值但不可避免的步骤**（1型浪费）
  3. **不创造价值且可以立即去除的步骤**（2型浪费）

### 原则3：流动（Make Flow）
- 让价值**不断流动**
- 消除阻碍流动的障碍
- 流动不是批量生产，是持续流动

### 原则4：拉动（Pull）
- **不要生产**，让客户**拉动**生产
- 客户需要时才生产，不需要就不生产
- 拉动 vs 推动（Push）：推动是按计划生产，拉动是按需求生产

### 原则5：尽善尽美（Seek Perfection）
- 永远没有"足够好"
- 持续改善，追求完美
- 完美不是终点，是方向

## 七种浪费（Muda）

| # | 浪费 | 含义 |
|---|------|------|
| 1 | **缺陷/返工** | 做错了要重做 |
| 2 | **过量生产** | 生产了客户不需要的 |
| 3 | **等待** | 等待下一个工序完成 |
| 4 | **运输** | 不必要的搬运 |
| 5 | **库存** | 积压的原料/在制品/成品 |
| 6 | **动作** | 不必要的身体动作 |
| 7 | **过度加工** | 超出客户需求的加工 |
| 8 | **人才浪费** | 没有利用人的智慧 |

## 核心工具

| 工具 | 用途 |
|------|------|
| **价值流图（VSM）** | 可视化整个价值流，识别浪费 |
| **单件流** | 一个一个流，不是批量 |
| **拉动式生产** | 按需求生产，不按计划生产 |
| **5S** | 整理/整顿/清扫/清洁/素养 |
| **改善（Kaizen）** | 小步快跑，持续改进 |
