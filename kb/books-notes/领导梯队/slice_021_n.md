# 领导梯队 — Slice 021 笔记：第12章 — 职能主管的发展路径

## 核心内容

### 职能主管的特殊路径

不是所有人都要走"事业部总经理→集团高管→CEO"这条路线。

### 两条常见路径

| 路径 | 说明 |
|------|------|
| **事业部路线** | 职能部门 → 事业部总经理 → 集团高管 → CEO |
| **职能路线** | 职能部门 → 高级职能专家 → 职能副总裁 |

### 职能路线的重要性

> "对大多数在大型公司工作的人来说，这是一个较可行的上升路径，因为被提升为事业部总经理的机会微乎其微。"

### 启示

不是每个人都要成为"全面的管理者"：
- 有些人是**深度专家**
- 有些人是**全面管理者**
- 两条路径都有价值

- 或者两条都可以？
