# 领导梯队 — Slice 011 笔记：第2章 — 从管理自我到管理他人

## 核心内容

### 第一次担任经理的挑战

> "第一次担任经理要想获得成功需要一个重大的转变——他们的工作成果不再是通过自己亲自做去获得，而是通过下属和团队的努力去获得。"

### 典型不适表现

- **心理上不适应**：还是想自己干活
- **行为上不适应**：事必躬亲，不授权
- **角色认知不清**：不知道自己的新职责

### 关键转变

| 之前 | 之后 |
|------|------|
| 自己做事 | 通过他人完成 |
| 个人贡献者 | 管理者 |
| 专家思维 | 管理思维 |

### 需要培养的能力
- 选拔人才
- 分配任务
- 教练辅导
- 绩效评估

## 可应用

阿山刚做管理时有没有这些问题？
- 事必躬亲？
- 不放心下属？
