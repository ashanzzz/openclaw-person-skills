# 领导梯队 — Slice 014 笔记：第5章 — 从管理职能部门到事业部总经理

## 核心内容

### 这是所有阶段中最大的挑战

> "事业部总经理将不得不改变他们的思维模式。"

### 事业部总经理的职责
- 全面负责产品或服务的研发、生产、营销
- 对成本与收入承担责任
- 有很强的掌控权

### 关键挑战

| 之前 | 之后 |
|------|------|
| 专注一个职能 | 全面负责 |
| 执行层 | 决策层 |
| 专业思维 | 商业思维 |

### 需要培养的能力
- 制定业务战略
- 全局思维
- 利润和成本意识
- 跨部门协调

### 典型问题
- 留恋过去的专业角色
- 不敢授权，还是想自己干
- 缺乏商业敏感度

阿山对照：老板或陈亮可能处于这个层级？
