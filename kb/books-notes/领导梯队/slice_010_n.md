# 领导梯队 — 章节笔记

## 六个领导阶段

### Stage 1: 管理自我 → 管理他人
- 从自己做事 → 通过他人完成工作
- 核心转变：学会激励和管理他人
- 常见错误：还是自己干活，不授权

### Stage 2: 管理他人 → 管理经理人员
- 从管几个人 → 管一群经理
- 核心：选拔和培养一线经理
- 常见错误：直接管一线员工，跳过经理

### Stage 3: 管理经理人员 → 管理职能部门
- 从管单一部门 → 管多个职能部门
- 核心：理解不同职能部门如何配合
- 常见错误：只管自己原来的专业领域

### Stage 4: 管理职能部门 → 事业部总经理
- 从单一产品/市场 → 整个业务单元
- 核心：制定战略、配置资源
- 常见错误：陷入日常运营，不做战略思考

### Stage 5: 事业部总经理 → 集团高管
- 从管一个业务 → 管多个业务
- 核心：筛选人才、配置资源、评估业务
- 常见错误：干预事业部日常运营

### Stage 6: 集团高管 → 首席执行官
- 从管理业务 → 管理整个公司
- 核心：文化、愿景、战略方向
- 常见错误：做太多具体决策，不做文化塑造

---

## 每阶段需要转变的三要素

| 要素 | 含义 |
|------|------|
| 领导技能 | 新岗位需要什么新能力 |
| 时间管理 | 如何分配时间 |
| 工作理念 | 什么是最重要的 |

---

## 常见问题

**拔苗助长**：
- 员工被提升到不胜任的位置
- 原因：晋升标准是业绩，不是管理能力

**事必躬亲**：
- 做了很多下属的工作
- 忽视了自己层级更重要的工作
