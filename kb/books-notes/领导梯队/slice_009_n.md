# 领导梯队 — Slice 009 笔记：导论

## 核心内容

### 领导力供需失衡

- 需求远大于供给
- 每期《华尔街日报》都有外部聘请高管新闻
- 猎头公司生意兴隆
- 咨询公司开出六位数起薪抢毕业生

### 外部招聘的问题

> "过于激进地试图从公司外部招聘人员的做法，表明公司内部人才管道存在缺陷。"

### 核心概念：领导梯队

- 不是"升职"而是"角色转变"
- 每一次晋升都是一次全新的领导技能、时间管理、工作理念的重新定位
- 层级越高，思维模式越需要改变

### 三维度

| 维度 | 说明 |
|------|------|
| 领导技能 | 该层级需要什么能力 |
| 时间管理 | 该层级如何分配时间 |
| 工作理念 | 该层级如何看待工作 |

## 可应用

阿山思考：
- 我现在在哪个层级？
- 这个层级需要什么技能？
- 我需要改变什么？
