# 领导梯队 — Slice 020 笔记：第11章 — 潜在缺陷

## 核心内容

### 领导力缺陷的原因

1. **被寄予厚望的人最终辜负期望**
2. **有人在第一层级就失败，有人到CEO层级才失败**
3. **培养和实际岗位需求脱节**

### 缺陷的常见模式

| 层级 | 常见缺陷 |
|------|---------|
| 第一层级 | 事必躬亲，不授权 |
| 第二层级 | 还像个人贡献者一样工作 |
| 第三层级 | 不愿意培养下属经理 |
| 第四层级 | 不放手，还管太细 |
| 第五层级 | 过于干预业务单元 |
| 第六层级 | 缺乏战略视角 |

### 关键洞察

> "导致缺陷的原因很多，而且其中一些是无法完全避免的。"

## 可应用

阿山对照：
- 我在哪个层级？有哪些潜在缺陷？
