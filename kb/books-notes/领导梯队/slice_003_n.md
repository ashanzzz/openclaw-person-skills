# 领导梯队 — Slice 003 笔记：目录

## 全书结构

**六个领导力发展阶段**

| 阶段 | 说明 |
|------|------|
| 第1阶段 | 从管理自我到管理他人 |
| 第2阶段 | 从管理他人到管理经理人员 |
| 第3阶段 | 从管理经理人员到管理职能部门 |
| 第4阶段 | 从管理职能部门到事业部总经理 |
| 第5阶段 | 从事业部总经理到集团高管 |
| 第6阶段 | 从集团高管到首席执行官 |

**三维度模型**
- 领导技能（skill）
- 时间管理（time application）
- 工作理念（work values）

**各阶段核心转变**
- 第1-3阶段：职能管理
- 第4-6阶段：战略管理
