# 驱动力 — Slice 010 笔记：第6章 — 目的

## 核心内容

### 目的的定义
做某件事是为了比自己更大的意义。

### 目的 + 自主 + 专精 的循环
```
有目的的人想自主
有自主的人想专精
有专精的人更理解目的
```

### 目的驱动的例子
- 维基百科：让知识自由共享
- 开源软件：让技术惠及所有人
- 诺贝尔和平奖得主穆罕默德·尤努斯的社会型企业

### 关键洞察
- 目的不是"利润最大化"
- 目的 = 超越自身，连接更大的意义
- 21世纪最成功的企业都有超越利润的目的

## 我的理解

阿山在工厂工作：
- 找到工作的意义："为什么做这件事比赚钱更重要"
- 和员工沟通时：说目的，不是说命令
- 目的 → 内在驱动 → 持久动力
