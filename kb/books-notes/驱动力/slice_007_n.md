# 驱动力 — Slice 007 笔记：第3章 — 做I型人还是X型人

## 核心内容

### X型人 vs I型人

| | X型 | I型 |
|--|-----|-----|
| 动力来源 | 外在奖励/惩罚 | 内在满足感 |
| 关注点 | 外部奖励 | 行为本身的成就感 |
| 表现 | 短期 | 持续 |
| 依赖 | 驱动力2.0 | 驱动力3.0 |

### 关键洞察

> "如果你相信大众皆平庸，平庸就会成为无法逾越的天花板。"

### I型的三个特征

1. **自主**（Autonomy）：自己决定做什么
2. **专精**（Mastery）：把想做的事做得越来越好
3. **目的**（Purpose）：超越自身的渴望

### 德西的遭遇
- 1971年德西在罗切斯特大学任职
- 1973年因"奖励观点离经叛道"被商学院开除
- 但他的研究揭示了：X型行为（外在驱动）是后天养成的

### 核心结论

- I型是**后天造就，不是天生注定**
- I型行为让：表现更佳 + 健康更好 + 幸福度更高
- 我们需要把自己和同事从X型变成I型

## 我的理解

阿山自我成长：
- 不要用外部奖励驱动自己
- 找到工作的内在满足感
- 追求专精：把事情做得越来越好本身就是奖励
