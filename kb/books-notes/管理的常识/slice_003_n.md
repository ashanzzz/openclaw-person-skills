# 管理的常识 — Slice 003 笔记：变革时代的管理

## 核心内容

### 变革无处不在
- 83%的CEO认为公司将在未来三年面临"重大"或"非常重大"变革
- 5年前这个比例只有65%
- 跟不上变革 = 被淘汰

### 变革的原因
1. **全球化** + 竞争激烈化
2. **技术快速变化**（互联网→移动互联网→AI）
3. **第三世界国家市场**快速发展

### 传统媒体案例（报纸行业）
- 《洛杉矶时报》《芝加哥论坛报》破产
- 《基督教科学箴言报》《西雅图信使报》停印
- 《纽约时报》向墨西哥首富求助
- 原因：没有跟上数字化变革

### 变革管理的核心

**IBM对1130名CEO的调查发现**：
- 变革不是意外，而是常态
- 跟上变革的速度本身就是竞争力
- 一名CEO形容："我的工作就是在沸水中漂流"

## 关键语录

> "变革无处不在，且来得既快又猛。"

## 可应用

- 公司面临变革 = 常态，不是意外
- 阿山面对陈亮/老板的决策变化 → 适应 + 找到新位置
- 个人成长 = 持续学习，跟上变化
