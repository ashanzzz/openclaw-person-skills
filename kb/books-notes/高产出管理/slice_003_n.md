# 高产出管理 — Slice 003 笔记：第2篇 — 管理杠杆率

## 核心内容

### 经理人的产出
> **经理人的产出 = 他所管辖单位的产出 + 他影响所及的其他单位的产出**

### 管理杠杆率
- **高杠杆率活动**：一次活动对团队产出有倍增效果
  - 培训、一对一、战略规划
- **中杠杆率活动**：一次活动影响多人
  - 开会、绩效评估
- **低杠杆率活动**：一次只影响一人
  - 回复邮件、具体事务

### 格鲁夫的洞察
- 经理人是"杠杆"，不是做事的人
- 通过他人完成工作才是经理人的产出
- 英特尔实际案例：主管芯片工厂的产出 = 良品率+质量

## 我的理解

阿山作为中层管理：
- 不要什么事都自己做
- 培养人 → 团队产出提升 → 阿山的产出提升
- 关注高杠杆率活动
