# 高产出管理 — Slice 006 笔记：第4篇 — 球员

## 核心内容

### 经理人是教练
- 经理人的产出 = 通过他人完成的工作
- 管理者 = 制定规则 + 创造环境 + 让球员发挥
- 不能上场替球员打球

### 绩效评估
- 评估 = 给员工真实反馈
- 不是年终打分，是持续的过程
- 标准要事先定好，不能事后评判

### 目标设定（OKR起源）
- **O** = Objective（目标）：鼓舞人心的方向
- **KR** = Key Results（关键结果）：可衡量的里程碑
- 关键：OKR 与绩效奖励**脱钩**

## 我的理解

阿山面对员工管理：
- 做好教练，不是做员工的替身
- 评估员工 → 给真实反馈，不是打分
- OKR → 让员工知道目标，不只是考核
