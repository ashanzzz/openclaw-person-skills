# 驱动力 — Slice 007 笔记：第3章 — I型人还是X型人

## 核心内容

### X型人 vs I型人

| | X型 | I型 |
|--|-----|-----|
| 动力 | 外在奖励/惩罚 | 内在满足 |
| 关注点 | 外部奖励 | 内在成就感 |
| 表现 | 短期 | 持续 |
| 驱动力 | 2.0 | 3.0 |

### 关键洞察
- "如果你相信大众皆平庸，平庸就会成为无法逾越的天花板"
- I型是后天造就，不是天生注定
- I型行为让表现更佳、健康更好、幸福度更高

## 可应用

阿山自我成长：
- 从X型转向I型
- 找到工作中的内在满足感，不只靠外部奖励/评价
