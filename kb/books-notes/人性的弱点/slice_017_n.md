# 人性的弱点 — Slice 017 笔记：第1篇第1章 — 想采蜜就请善待蜂巢

## 核心内容

### 核心法则1：不要批评、指责或抱怨

**关键洞察**：
- 99%的人，无论做错什么事情，**都不会批评自己**
- 阿尔·卡彭、"双枪手"科洛利：即使临死前也认为自己是好人
- 批评不能让对方改变，反而招来怨恨

### 实验证明
- B.F.斯金纳：奖励让动物学好比惩罚让动物改错**更有效**
- 哈佛商学院：批评和指责不能改善情况，反而让人辩护和反批评

### 经典案例
- 林肯：年轻时喜欢写信讽刺别人 → 一次差点决斗 → 从此再也不用刻薄语言
- 林肯写给米德将军的信（批评贻误战机）：写完但**从未寄出** → 从遗物中发现
- 西奥多·罗斯福批评塔夫脱 → 共和党灾难性分裂 → 塔夫脱含泪辩护

### 林肯的转变
林肯的改变：他仰视白宫墙上的林肯画像，自问"如果林肯遇到这事会怎么做？"
→ 理解 > 指责

## 我的理解

### 工厂场景

**阿山的情况**：
- 老板/陈亮经常批评员工 → 员工防御性辩解 → 问题没解决
- 正确做法：理解对方为什么这么做，而不是直接批评

**安全帽案例**：
- ❌ "不戴安全帽就罚款！" → 员工表面服从，离开就摘
- ✅ "帽子戴着不舒服吗？" → 员工主动戴
