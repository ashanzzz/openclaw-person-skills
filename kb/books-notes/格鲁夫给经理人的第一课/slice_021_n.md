# 格鲁夫给经理人的第一课 — Slice 021 笔记：第10章 — 三个"长官"

## 核心内容

### 员工听命的三个控制模式

| 控制模式 | 说明 |
|---------|------|
| **自己管理** | 自我驱动，有内在目标 |
| **同伴压力** | 同伴影响行为（团队规范）|
| **上级管理** | 外部监督和评价 |

### 关键洞察

> "控制模式必须因时制宜，一药不能治百疾。"

### 经理人如何选择
- 高成熟度员工 → 自己管理
- 新员工 → 上级管理
- 团队文化强 → 同伴压力

## 可应用

阿山管员工：
- 老员工 → 给自主空间
- 新员工 → 给明确指导
- 团队文化 → 让同伴相互影响
