# 格鲁夫给经理人的第一课 — Slice 027 笔记：第15章 — 报酬

## 核心内容

### 金钱的双重角色

| 角色 | 说明 |
|------|------|
| **保健因素** | 基本工资 → 不够会不满 |
| **激励因素** | 绩效工资 → 够多也不一定激励 |

### 报酬原则

1. **报酬要明显反映绩效**
2. **高于市场平均水平**（吸引人才）
3. **内部公平**：同岗不同酬要解释

### 关键洞察

> "金钱在马斯洛的所有激励层次中都扮演着相当重要的角色。但当需求层次向上提升后，金钱便从衣食温饱转变为衡量竞争价值的标准。"

## 可应用

阿山发工资：
- 基本工资要够（否则员工不满）
- 绩效奖励要让员工看得见
- 不是发越多越好，是发得让人感觉公平
