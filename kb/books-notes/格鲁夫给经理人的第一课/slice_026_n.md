# 格鲁夫给经理人的第一课 — Slice 026 笔记：第14章 — 招人与留人

## 核心内容

### 面试的四个目的

| # | 目的 |
|---|------|
| 1 | 找到具备能力及潜力的人 |
| 2 | 让他认识你和公司 |
| 3 | 确定能力和工作是否匹配 |
| 4 | 把他招进公司 |

### 留人的关键
- 了解员工真正在乎什么
- 提供挑战和成长机会
- 不只是谈钱

### 关键洞察

> "一个经理人迟早都会碰到两个任务：面试新进员工，以及设法让一个萌生去意的优秀部属继续待在公司效命。"

## 可应用

阿山招人：
- 问：你最在乎什么？
- 看：这个人能不能成长
- 留：用挑战，不只用钱
