# 格鲁夫给经理人的第一课 — Slice 014 笔记：第4章 — 开会是必经之路

## 核心内容

### 开会的现实
- 经理人超过50%时间在开会
- 德鲁克：如果超过25%时间在开会，组织大概有问题

### 两种会议类型

| 类型 | 说明 |
|------|------|
| **过程协调会议** | 例行，有固定议程（如早会） |
| **解决问题会议** | 临时，针对特定问题 |

### 让会议更有效的原则
1. 准时开始，准时结束
2. 有明确议程
3. 明确决策人和行动项
4. 会后跟进

## 可应用

阿山开早会：
- 控制时间
- 有议程
- 明确谁负责什么
