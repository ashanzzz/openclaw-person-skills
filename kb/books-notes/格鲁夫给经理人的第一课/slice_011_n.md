# 格鲁夫给经理人的第一课 — Slice 011 笔记：第2章 — 早餐店库存

## 核心内容

### 规模扩大后的管理挑战
- 单一早餐店 → 连锁体系
- 需要一套**衡量指标**
- 产出 = 整个工厂的产品 + 利润 + 顾客满意度

### 关键概念

**设定指标的重要性**：
- 衡量指标决定管理重点
- "如果你无法衡量它，你就无法管理它"
- 早餐店经理的产出是整个工厂，不是自己干活

### 库存管理
- 批量生产的弊端：库存掩盖问题
- 库存增加 = 资金占用 + 质量隐患
- 精益生产的关键：减少库存

## 可应用

阿山在工厂：
- 进盛/打磨工厂的产出是什么？
- 用什么指标衡量？
- 不要让库存掩盖工序问题
