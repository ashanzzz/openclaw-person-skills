# 格鲁夫给经理人的第一课 — Slice 013 笔记：第3章 — 管理杠杆率

## 核心内容

### 经理人的产出
> **经理人的产出 = 他所管辖单位的产出 + 他影响所及的其他单位的产出**

### 经理人的活动例子
- 判断与意见
- 执行方向
- 资源分配
- 检查错误
- 培训与发展部属
- 传授课程
- 产品计划
- 沟通协调

### 管理杠杆率
- **高杠杆率活动**：一次活动多次影响（培训、一对一）
- **中杠杆率活动**：一次活动影响多人（开会）
- **低杠杆率活动**：一次只影响一人（具体事务）

### OKR起源
- **O** = Objective（目标）：鼓舞人心的方向
- **KR** = Key Results（关键结果）：可衡量的里程碑
- 与绩效奖励**脱钩**：保持探索空间

## 可应用

阿山作为中层：
- 不要什么事都自己做（低杠杆）
- 培养人是最高杠杆率活动
