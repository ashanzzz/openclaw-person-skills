# 格鲁夫给经理人的第一课 — Slice 019 笔记：第8章 — 混血型组织

## 核心内容

### 两种极端组织形态

| 类型 | 说明 |
|------|------|
| **任务导向** | 每个分店独立运营，总公司只管财务 |
| **功能导向** | 所有分店统一采购、人事、营销 |

### 大多数组织是"混血型"
- 介于两种极端之间
- 需要平衡：地方灵活性 vs 集中资源共享
- 中层经理是关键协调人

### 关键洞察

> "大部分中层经理人所负责的，都只是大组织中的一个部门。他们能监控的'黑箱'与其他不同的黑箱紧密相连。"

## 可应用

吉众/祺富两工厂：
- 既要有各自灵活性，也要有统一协调
- 阿山的位置 = 协调两厂的中层
