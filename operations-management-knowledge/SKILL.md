# Operations & Management Knowledge Base

整合三本经典管理著作，为工厂、公司及个人提供可执行的 SOP 和决策框架。

## 三本书的核心价值

| 书籍 | 核心贡献 | 解决什么问题 |
|------|---------|------------|
| **《目标》(TOC)** | 瓶颈识别 + 约束理论 | 工厂效率低、产出慢、不知从哪下手 |
| **《丰田之道》** | 现场管理 + 持续改善 | 改善推不动、标准难建立、问题反复 |
| **《高效能人士七个习惯》** | 个人效能 + 团队协作 | 员工不主动、团队内耗、方向迷失 |

## 三步决策路径

```
第一步：识别真正的问题
↓
TOC说：找到瓶颈（不要猜测，要数据）
↓
第二步：选择工具
↓
问题在系统/流程？→ TOC五步法 + 丰田DBR
问题在人/团队？→ 七习惯框架
问题在文化/标准？→ 丰田14原则
↓
第三步：执行并标准化
↓
改善 → 标准化 → 再改善（持续循环）
```

## 关键原则速查

### TOC 五步法（找瓶颈）
1. 识别瓶颈
2. 挖尽瓶颈
3. 一切服从瓶颈
4. 提升瓶颈
5. 重复循环

### 丰田之道四柱
1. 持续改善（Kaizen）
2. 尊重人性（Jin-kyo-jin）
3. 现地现物（Genchi Genbutsu）
4. 拉式生产（Pull）

### 七习惯成长路线
```
个人领域（Private Victory）
习惯一：积极主动
习惯二：以终为始
习惯三：要事第一

公众领域（Public Victory）
习惯四：双赢思维
习惯五：知彼解己
习惯六：统合综效

持续更新
习惯七：不断更新
```
