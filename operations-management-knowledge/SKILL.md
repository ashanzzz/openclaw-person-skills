# Operations & Management Knowledge Base

整合九本经典管理著作，为工厂、公司及个人提供可执行的 SOP 和决策框架。

## 九本书的核心价值

| 书籍 | 核心贡献 | 解决什么问题 |
|------|---------|------------|
| **《目标》(TOC)** | 瓶颈识别 + 约束理论 | 工厂效率低、产出慢、不知从哪下手 |
| **《精益思想》** | 七种浪费 + 五大原则 | 流程中的浪费识别与消除 |
| **《丰田之道》** | 现场管理 + 持续改善 | 改善推不动、标准难建立、问题反复 |
| **《高效能人士七个习惯》** | 个人效能 + 团队协作 | 员工不主动、团队内耗、方向迷失 |
| **《驱动力》** | 自主+专精+目的 | 内在动机激活、外在奖励失效 |
| **《人性的弱点》** | 人际沟通六原则 | 关系处理、批评/赞赏、说服 |
| **《格鲁夫给经理人的第一课》** | 管理杠杆率 + OKR起源 | 中层管理输出最大化、OKR |
| **《领导梯队》** | 六阶段领导力发展 | 人才晋升错位、梯队阻滞、越俎代庖 |
| **《第五项修炼》** | 系统思考 + 五项修炼 | 复杂问题诊断、组织学习 |

## 五步决策路径

```
第一步：识别问题类型
↓
工厂效率低/瓶颈？→ TOC/精益
改善推不动/现场乱？→ 丰田之道
员工不主动/无激励？→ 驱动力
人才梯队乱/越俎代庖？→ 领导梯队
沟通障碍/人际冲突？→ 人性的弱点
中层管理输出低？→ 格鲁夫
复杂系统问题？→ 第五项修炼
↓
第二步：选择工具/框架
↓
第三步：执行并标准化
↓
第四步：持续改善（回到第一步）
```
