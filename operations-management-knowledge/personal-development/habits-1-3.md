# 个人发展：习惯 1-3（个人效能）

---

## 习惯一：积极主动（Be Proactive）

### 核心：人有选择回应的自由

```
外部刺激（不管多恶劣）
    ↓
（每个人都有选择的自由）
    ↓
回应方式
```

### 积极主动的语言转换

| 消极被动 | 积极主动 |
|---------|---------|
| "他让我很生气" | "我可以改变自己的反应" |
| "我也没办法" | "我试试其他方法" |
| "他们肯定不会同意" | "我可以先听听他们的顾虑" |
| "我只能这样了" | "还有其他可能性" |

### 个人应用

```
停止说"我必须/不得不"
改为"我选择……因为……"

停止说"他太过分了"
改为"我可以选择理解他的处境"

停止说"没办法"
改为"我需要更多信息，还有什么可能性？"
```

## 习惯二：以终为始（Begin with the End in Mind）

### 核心：先定目标，再行动

### 个人使命宣言模板

```
我是谁？
我代表什么？
我的人生目标是什么？
关系目标是什么？
职业目标是什么？
```

### 角色平衡

每个人有多个角色，失衡会出问题：

```
                    精神/使命
                       ↑
                      习惯二
健康 ←——————————→ 财务
                      习惯三        习惯一：积极主动
                       ↓             习惯四：双赢思维
                    角色              习惯五：知彼解己
                   /  | \
              工作  家庭  朋友         习惯六：统合综效
```

### 以终为始的决策标准

```
任何决策前先问：
1. 我想要什么结果？
2. 这个决定是否指向我的目标？
3. 如果这是我人生的最后一年，我会做这个决定吗？
4. 这个决定是否符合我的价值观？
```

## 习惯三：要事第一（Put First Things First）

### 核心：做重要的事，不只是做紧急的事

### 紧急/重要矩阵

|  | **紧急** | **不紧急** |
|--|---------|----------|
| **重要** | 第一象限（危机/紧迫Deadline） | **第二象限（真正重要）** |
| **不重要** | 第三象限（干扰/他人紧急） | 第四象限（浪费时间） |

### 典型时间分配

| 象限 | 普通人 | 高效能人士 |
|------|--------|-----------|
| 第一象限 | 60% | 15% |
| 第二象限 | 5% | **65%** |
| 第三象限 | 25% | 15% |
| 第四象限 | 10% | 5% |

### 第二象限的日常实践

```
每天第一件事：先做第二象限工作
不要以紧急事务开始一天

每周预留"不受干扰时间"：
- 2小时战略规划
- 2小时与核心人1对1
- 1小时学习/自我提升
```

### 授权的艺术

```
不授权（自己做）：
- 紧急但不重要（第三象限）

有效授权（培养人）：
- 需要培养他人的能力（第二象限，长期）

充分授权（完全信任）：
- 对方完全能胜任，且你完全信任
```

## P/PC Balance（产出与产能平衡）

### 核心公式

```
效能 = 产出（P）/ 产能（PC）
```

### 杀鸡取卵 vs 养鸡生蛋

```
杀鸡取卵（只追求产出）：
- 只看眼前收入，不保养设备
- 员工过度疲劳，没有培训
- 透支身体，只顾工作
结果：系统崩溃，产出归零

养鸡生蛋（保护产能）：
- 定期维护设备（保护PC）
- 员工培训成长（保护PC）
- 保持工作和健康平衡（保护PC）
结果：可持续的高产出
```

### 工厂/工作中的应用

| 场景 | P（产出） | PC（产能） |
|------|---------|-----------|
| 工厂 | 订单交付 | 设备保养、员工培训 |
| 销售 | 签单 | 客户关系维护 |
| 管理 | 业绩 | 团队能力成长 |
| 个人 | 工作成果 | 身体/精神/智力/社交 |


> **原则：透支产能追求产出，最终两败俱伤。**
