# 精益思想（Lean Thinking）

> **作者**：詹姆斯·沃麦克（James P. Womack）和丹尼尔·琼斯（Daniel T. Jones）
> **出版**：1996年
> **核心**：精益生产的五大原则，丰田生产系统（TPS）的精华提炼
> **适用**：制造业、服务业、企业流程改善

---

## 一、精益的两大核心原则

### 1. 消除浪费（Muda）

**浪费定义**：任何不增加客户价值的活动。

### 2. 尽善尽美（Kaizen）

**尽善尽美定义**：永不满足现状，持续改善。

---

## 二、七种浪费（Muda）

| 序号 | 浪费类型 | 说明 | 工厂例子 |
|------|---------|------|---------|
| 1 | 等待 | 物料/人员/设备等待 | 机器故障后等待维修 |
| 2 | 搬运 | 不必要的物料移动 | 大量在制品库存导致多次搬运 |
| 3 | 缺陷 | 废品/返工 | 产品不良导致返工 |
| 4 | 过度生产 | 超过客户需要的数量 | 提前生产导致库存堆积 |
| 5 | 库存 | 超过最小需要的库存 | 大量原料/在制品/成品库存 |
| 6 | 过度加工 | 超出客户需求的加工 | 精度要求超出客户要求 |
| 7 | 动作 | 不必要的人员动作 | 走动、弯腰、伸手等 |

### 额外浪费：人才的浪费

第8种浪费：未能充分发挥员工的智慧和能力。

## 三、精益五大原则

### 原则一：价值（Value）

**定义**：只有客户愿意为之付钱的活动才创造价值。

**关键问题**：
```
问：这项活动对客户有价值吗？
→ 是：保留
→ 否：消除
```

**客户价值的特征**：
- 客户想要
- 第一次就正确
- 在正确的时间
- 在正确的地点
- 以正确的价格

### 原则二：价值流（Value Stream）

**定义**：从原材料到交付给客户的全部活动，包括：
- 增值活动
- 不增值但不必要活动（可接受）
- 纯粹浪费

**价值流分析步骤**：

```
步骤1：绘制当前状态图（Current State Map）
步骤2：识别浪费
步骤3：绘制未来状态图（Future State Map）
步骤4：实施改善
```

### 原则三：流动（Flow）

**定义**：让价值顺畅地流动，不停滞、不等待。

**实现流动的障碍**：
- 批量思维（大批量生产）
- 部门壁垒
- 质量不稳定
- 设备故障
- 等待时间

**流动的实现方法**：
```
1. 连续流（Continuous Flow）：工序之间无缝连接
2. 单元生产（Cell Production）：U型单元布局
3. 快速换模（SMED）：换型时间<10分钟
4. 全面生产维护（TPM）：设备零故障
```

### 原则四：拉动（Pull）

**定义**：后工序只在需要时才向前工序领取物料。

**拉动原则**：
```
推式（Push）：计划生产多少就生产多少
拉式（Pull）：客户需求触发生产

看板（Kanban）是拉动的工具：
1. 后工序向前提取信号
2. 前工序按需补充
3. 不多不少，刚刚好
```

**拉动的好处**：
- 库存最小化
- 问题立即暴露（库存掩盖问题）
- 响应市场需求变化

### 原则五：尽善尽美（Perfection）

**定义**：
```
消除浪费 → 价值流动 → 实现拉动 → 发现更多浪费 → 持续改善

尽善尽美 = 永无止境的改善
```

## 四、精益工具详解

### 1. 5S

| S | 日文 | 中文 | 说明 |
|---|------|------|------|
| 1S | Seiri | 整理 | 区分要/不要 |
| 2S | Seiton | 整顿 | 定位、定量、定标准 |
| 3S | Seiso | 清扫 | 清除灰尘/污物 |
| 4S | Seiketsu | 清洁 | 标准化、制度化 |
| 5S | Shitsuke | 素养 | 习惯成自然 |

### 2. 看板（Kanban）

**看板规则**：
```
规则1：后工序向前提取
规则2：前工序只生产被领取的数量
规则3：不合格品不传递给下工序
规则4：看板数量最小化
规则5：看板用于管理，不是计数
```

**看板数量计算**：
```
看板数量 =（日均需求量 × 交付时间）+ 安全库存
日均需求：200件/天
交付时间（补货周期）：1天
安全库存：10%
看板数量 = 200 × 1 × 1.1 = 220件
```

### 3. 快速换模（SMED）

**目标**：换型时间 < 10分钟

**SMED三步法**：
```
步骤1：区分内部换型（停机时）和外部换型（不停机时）
步骤2：把内部换型转为外部换型（准备可以在运行中做）
步骤3：优化所有换型活动
```

**案例**：注塑机换模 4小时 → 30分钟
```
原来：停机后找工具、调试、再调试
改善后：工具预热、参数标准化、快速夹具
```

### 4. 全面生产维护（TPM）

**目标**：零故障、零缺陷、零事故

**TPM八支柱**：
```
1. 自主维护（操作员日常点检）
2. 计划维护（定期保养）
3. 个别改善（KP问题攻关）
4. 设备初期管理（新设备导入）
5. 质量维护（质量隐患预防）
6. 安全/环境（零事故、零灾害）
7. 教育训练（技能提升）
8. TPM日常工作（现场管理）
```

### 5. 单元生产（Cell Production）

**U型单元布局**：
```
投入 → [工序1] → [工序2] → [工序3] → 产出
            ↑                    ↓
            ←←←←←←←←←←←←←
                  返回通道
```

**单元生产的优势**：
- 一个人完成多道工序（多技能工）
- 减少搬运
- 缩短交付周期
- 问题立即可见

## 五、精益实施路线图

### 阶段一：现场基础（0-3个月）

```
目标：建立精益基础
行动：
1. 5S 实施
2. 标准化作业
3. 班组日常管理
4. 快速问题响应机制
```

### 阶段二：连续流建立（3-6个月）

```
目标：消除等待和搬运
行动：
1. 工序重新布局（Cell化）
2. 小批量生产
3. SMED 实施
4. TPM 启动
```

### 阶段三：拉动系统（6-12个月）

```
目标：建立拉动，减少库存
行动：
1. 看板系统导入
2. 供应商拉动补料
3. 客户需求拉动
4. 库存最小化
```

### 阶段四：持续改善（12个月+）

```
目标：建立改善文化
行动：
1. 改善周（Kaizen Week）
2. 价值流改善
3. 精益供应链
4. 精益设计/研发
```
