# Completions Log — Template

Append to `~/work-tracker/completions.md`.

```markdown
## YYYY-MM-DD

- [#id] "Task title" — project: {project_name}
  Completed at: HH:MM Beijing
  Context: what this was about / what led to it
  Follow-up needed: yes → [what] / no / maybe → [what to watch for]

- [#id] "Task title" — project: {project_name}
  ...
```

## Guidelines

- Log immediately when task marked done in Vikunja
- Be specific: "finished feature X" not just "worked on project"
- Note context for future pattern learning: "completing this revealed Y needs testing"
- Mark follow-up clearly: yes/no/maybe with specific item if known
