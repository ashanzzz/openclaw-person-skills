# 管理学：七习惯框架

## 成长路线图

```
依赖期（靠别人）→ 独立期（靠自己）→ 互赖期（靠团队）

个人领域（Private Victory）
├── 习惯一：积极主动
├── 习惯二：以终为始
└── 习惯三：要事第一

公众领域（Public Victory）
├── 习惯四：双赢思维
├── 习惯五：知彼解己
├── 习惯六：统合综效
└── 习惯七：不断更新
```

---

## 习惯一：积极主动

**人有选择回应的自由**。外部刺激 →（选择）→ 回应

### 管理要点

| 消极被动的管理者 | 积极主动的管理者 |
|----------------|----------------|
| "老板就这样，我能怎么办" | "我可以选择更好的回应方式" |
| "员工不配合，没办法" | "我可以创造让他们配合的条件" |
| 抱怨问题 | 聚焦自己能影响的事 |

### 实践

```
判断维度：这件事我能控制吗？
能控制   → 立即行动
能影响   → 选择最佳影响方式
不能控制 → 接受，或转移到影响圈
```

---

## 习惯二：以终为始

**先定目标，再行动**。以原则为中心，不是以角色/情绪/习惯为中心。

### 管理要点

```
任何管理决策前先问：
1. 我想要什么结果？
2. 这个决定是否指向我们的目标？
3. 如果这是我任期的最后一天，会做这个决定吗？
4. 这个决定是否符合公司价值观？
```

---

## 习惯三：要事第一

**做重要的事，不只是做紧急的事**。

### 紧急/重要矩阵（管理版）

| 象限 | 内容 | 正确做法 |
|------|------|---------|
| **第二象限** | 设备预防性维护、工艺改进、员工培训 | **优先安排** |
| 第一象限 | 机器故障停线、质量事故 | 救火，但不能只做这个 |
| 第三象限 | 同事突然求助、过多会议、老板临时安排 | 学会说"不"或"稍后" |
| 第四象限 | 无效会议、刷手机、过度社交 | 消除 |

### 授权原则

```
不授权（自己做）：
- 紧急但不重要的事

有效授权（培养人）：
- 需要培养他人的能力（第二象限，长期）

充分授权（完全信任）：
- 对方完全能胜任
```

## 习惯四：双赢思维

**所有合作都寻求双方利益最大化**。

### 工厂/公司双赢应用

| 关系 | 错误思维 | 正确思维 |
|------|---------|---------|
| 供应商 | "我压价你供货"（零和博弈） | "帮助供应商改善，双方受益" |
| 员工 | "我发工资你干活"（交易型） | "帮助员工成长，公司也受益" |
| 客户 | "卖出去就不管了" | "帮客户解决问题，建立长期信任" |
| 部门 | "我只管自己部门的KPI" | "整体最优才是真的好" |

## 习惯五：知彼解己

**先理解别人，再求被理解**。

### 沟通错误 vs 正确

```
❌ 大多数管理者：
听到员工说话 → 想自己怎么回答 → 员工停就发表意见

✅ 真正的管理者：
先暂停自己想说的欲望
→ 用心听员工说了什么
→ 确认理解正确（"你是说……对吗？"）
→ 再表达自己的意见
```

### 同理心四步

```
1. 复述对方的话（"你是说……对吗？"）
2. 解释对方行为背后的原因
3. 表达对对方情感的理解（"这确实让人很为难/沮丧"）
4. 有条件地询问（"我想多了解一些……可以吗？"）
```

## 习惯六：统合综效

**整体大于部分之和**。差异是资产，不是威胁。

### 第三选择思维

```
第一选择：我的方案
第二选择：你的方案
第三选择：我们的方案（比前两者都好）

寻找第三选择四步：
1. 双方各自充分表达
2. 明确双方真正想要的是什么（超越立场）
3. 头脑风暴，寻找新的可能性
4. 找到双方都真正愿意执行的方案
```

---

## 习惯七：不断更新

**保持四个维度的平衡发展**。

| 维度 | 含义 | 管理实践 |
|------|------|---------|
| 身体 | 健康/体能 | 合理作息，不过度消耗 |
| 精神 | 价值观/使命感 | 定期回顾使命宣言 |
| 智力 | 学习/思考/规划 | 持续学习、阅读、反思 |
| 社交 | 人际关系 | 维护核心关系 |

### P/PC Balance（产出与产能平衡）

```
杀鸡取卵：只追求产出（P），透支产能（PC）
→ 机器不保养，员工过度疲劳，系统崩溃

养鸡生蛋：保护产能，稳定产出
→ 可持续的高产出
```

## 决策矩阵（管理场景应用）

| 场景 | 核心习惯 | 行动 |
|------|---------|------|
| 员工不主动，等安排 | 习惯一（积极主动） | 建立"主动改善"的激励机制 |
| 不知道往哪走 | 习惯二（以终为始） | 明确部门/个人使命宣言 |
| 总被紧急事打断 | 习惯三（要事第一） | 每天留2小时做第二象限工作 |
| 供应商总压不动 | 习惯四（双赢思维） | 建立年度改善合作机制 |
| 下属抱怨不听 | 习惯五（知彼解己） | 同理心倾听，不要给建议 |
| 部门各自为战 | 习惯六（统合综效） | 跨部门项目，找第三选择 |
| 团队疲惫无进步 | 习惯七（不断更新） | 培训/轮岗/挑战性任务 |
