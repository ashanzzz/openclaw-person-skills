# 驱动力（Drive）

> **作者**：丹尼尔·平克（Daniel H. Pink）
> **出版**：2009年（全球销量超过200万册）
> **核心**：揭示真正驱动人类行为的动机——不是金钱奖励，而是**自主、专精、目的**
> **副标题**：关于激励的惊人真相

---

## 一、核心论点：胡萝卜加大棒失灵了

传统激励逻辑（奖励/惩罚 = 外在动机）：
- 适合简单、重复机械性任务
- **不适合**复杂、需要创意、概念性任务

**研究证据**：
- 心理学家萨姆·格洛钦的"20问题"实验：奖励越多，表现越差
- 诺贝尔奖得主丹尼尔·卡尼曼：贫困时金钱有用，到一定程度后，边际效益递减

**结论**：对于需要"认知"（思考、分析、创造）的工作，传统的"胡萝卜加大棒"反而有害。

---

## 二、三大驱动力

### 第一驱动力：生理需求（必须满足，但不是动机）

满足生存的基本需求（水、食物、住所）。

### 第二驱动力：外在动机（奖励/惩罚）

"做X就奖励Y"、"做错就惩罚"。适用于：
- 目标明确
- 路径清晰
- 不需要创意

**不适用于**：需要判断、创意、解释模糊情况的场景。

### 第三驱动力：内在动机（真正的驱动力）

做某件事本身就是奖励，不靠外在奖励驱动。

---

## 三、内在动机的三根支柱

### 支柱一：自主（Autonomy）

**定义**：自己决定做什么、何时做、如何做、与谁一起做。

自主不是"独立工作"，而是**对自己生活的掌控感**。

**实践案例**：

| 公司 | 自主实践 |
|------|---------|
| Google | 20%时间做自己想做的项目（Gmail/Android来源） |
| Atlassian | "ShipIt"日：每季度一天，员工做任何想做的 |
| 3M | 15%时间做任何项目（Post-it来源） |
| Zappos | 自主决定工作方式 |

**在管理中的应用**：
```
不是"你必须做什么"（控制）
而是"你能做什么"（自主）

给予员工：
- 工作内容的自主（做什么任务）
- 工作时间的自主（弹性时间）
- 工作方法的自主（怎么完成）
- 工作团队的自主（和谁合作）
```

### 支柱二：专精（Mastery）

**定义**：把某件事做得越来越好的欲望。

**专精的三个规律**：

1. **专精是心态**：相信能力是可以发展的（成长型思维 vs 固定型思维）
2. **专精是痛苦的**：在"舒适区"之外工作（刻意练习）——但不能太痛苦，否则会放弃
3. **专精是曲线**：快速进步 → 平台期 → 下一个进步

**刻意练习的要素**：
```
1. 专注于特定技能
2. 立即反馈
3. 专注于改进
4. 离开舒适区
5. 重复，重复，再重复
```

**在管理中的应用**：
```
- 给员工持续成长的机会（不是"做完就完了"）
- 提供立即反馈（不要等年度评估）
- 让员工挑战略高于自己能力的工作（flow状态）
- 允许失败是学习过程的一部分
```

### 支柱三：目的（Purpose）

**定义**：做某件事是为了比自己更大的意义。

目的驱动的组织：
- 不是"利润最大化"，而是"让世界更美好"
- 利润是做好事情的结果，不是目标

**目的 + 自主 + 专精 的组合**：
```
有目的的人想自主（为了实现目的）
有自主的人想专精（为了更好实现目的）
有专精的人更理解目的（专精带来更深理解）
```

## 四、Type X vs Type I 行为

| | Type X（外在驱动） | Type I（内在驱动） |
|--|-------------------|-------------------|
| 假设 | 人天生懒惰，需要控制 | 人天生有成长欲望 |
| 动力 | 外部奖励/惩罚 | 内在满足感 |
| 目标 | 短期结果 | 长期成长 |
| 对奖励 | 依赖 | 不依赖 |

**关键洞察**：Type I 不是"不在乎奖励"，而是"在满足基本需求后，奖励的作用递减"。

---

## 五、动机的演变

```
农业时代 → 工业时代 → 信息时代
  生理    外在奖励   内在动机
  需求      (Type X)   (Type I)
```

在**信息时代**，真正有价值的资产是：
- 知识
- 创造力
- 自主判断

这些都需要**内在动机**来驱动，外部奖励反而会扼杀。

## 六、绩效评估的重新设计

### 传统方式（外在动机）
```
KPI → 达成奖励/未达成惩罚
```
问题：关注数字，不关注价值；短视；扼杀创新

### 新方式（内在动机）
```
OKR（目标与关键结果）
- 目标（Objective）：方向性的，有抱负的
- 关键结果（Key Results）：可衡量的，具体指标
- 不直接与奖励挂钩（保持探索空间）
```

## 七、对管理的实践启示

### 给予自主的七条原则

1. **开始提问，而不是下达命令**
2. **承认"我已经知道"是进步的最大敌人**
3. **让员工自己设定目标**（与下属一起，不是单方面）
4. **用"检查"代替"控制"**（定期review，而不是微观管理）
5. **奖励好奇**，而不是惩罚犯错
6. **让员工知道"为什么"**，而不只是"做什么"
7. **允许追求专精的过程中的"低谷"**

### 营造专精环境的三步

```
1. 设立"难度略高"的任务（踮脚够得到）
2. 提供即时反馈（不要等很久才告诉员工）
3. 允许重复和练习的时间
```

### 激发目的感的两个方法

```
1. 让工作与更大的意义连接
   "我们在卖保险" → "我们在保护人们的安全感"

2. 让员工参与目标设定
   参与感 = 目的感
```
